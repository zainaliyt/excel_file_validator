# CPS System Project

This project was made for CPS System. It provides a simple interface for uploading and validating Excel files, with a history of uploads displayed in real-time.

## Simple Guide

To successfully run this project locally, follow the steps below:

### Install Packages

1. **Node Packages**
   - Navigate to the project directory and run the following command to install the required Node packages:
     ```bash
     npm install
     ```

2. **Python Packages**
   - Ensure you are using Python 3.9. Then, install the necessary Python packages:
     ```bash
     pip install flask flask_cors pandas
     ```

### Run the Project

1. **Start the Web App**
   - Run the following command to start the React web application:
     ```bash
     npm start
     ```

2. **Start the Python Script**
   - In a separate terminal, run the Python script to start the backend server:
     ```bash
     python backend.py
     ```

### Testing

- **Excel Files**
  - Test the functionality using the provided Excel files or use your own. Ensure the files meet the validation criteria specified in the project.

## Note

- **Python Version**
  - Make sure you are using Python 3.9 to avoid any compatibility issues.
